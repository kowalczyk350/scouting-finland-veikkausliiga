# Skauting i Obserwacja - Ligi Fińskie

Projekt poświęcony analizie danych oraz obserwacji zawodników występujących w fińskich ligach piłkarskich (głównie Veikkausliiga).

## Kryteria Skautingowe (Target):
* **Wiek:** 22-23 lata.
* **Profil:** Zawodnicy o wyróżniających się parametrach fizycznych, połączonych z doskonałym wyszkoleniem technicznym.
* **Finanse:** Zawodnicy z kończącymi się kontraktami lub relatywnie niską ceną rynkową (w zasięgu finansowym klubów Ekstraklasy / 1. Ligi).
* **Cel:** Identyfikacja graczy z Veikkausliigi gotowych do rywalizacji na wyższym poziomie ligowym.
